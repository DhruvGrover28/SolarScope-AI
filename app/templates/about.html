Quick start

  1. Pick your analysis method from the dropdown.
  2. Fill the project name and required inputs.
  3. Submit to generate the rooftop estimate.
  4. Download the report from the project page.

Method guide

  • Upload image: Best for clear satellite or drone images.
  • Address: Uses OpenStreetMap footprints.
  • Manual area: Use when you already know usable roof area.
  • Draw on map: Trace the roof outline directly.

Troubleshooting

  • If address lookup fails, try a nearby landmark.
  • For map drawing, zoom in for better accuracy.
  • Upload images in JPG or PNG format.
  • If the mask looks off, try a clearer top-down image.
